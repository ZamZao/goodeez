# INSTRUCTIONS COPILOT – STYLE & COPYWRITING (FR)

## 1. Style d’écriture global
Lorsque tu génères du texte ou du copywriting :

- Ton : professionnel, chaleureux, clair, orienté RH / People / Office Managers.
- Style : concis, direct, inspiré des leaders du marché (SwagUp, Printfection), mais adapté au contexte français.
- Évite le jargon marketing inutile.
- Écris des phrases courtes, percutantes, orientées bénéfices.
- Toujours mettre en avant : gain de temps, logistique externalisée, expérience premium pour les collaborateurs, simplicité, fluidité.

## 2. Guidelines visuelles (Next.js + Tailwind)
Dans le code JSX/TSX, respecter :

- **Tailwind** obligatoire.
- Largeur max : `max-w-6xl mx-auto px-6`.
- Espacements généreux entre les sections : `py-20`.
- Composants visuels avec :
  - `shadow-lg`
  - `border border-gray-100`
  - `rounded-2xl`
  - `bg-white`
- Grilles :
  - `grid grid-cols-1 md:grid-cols-2 gap-12`
- Layouts simples :
  - `flex flex-col gap-8`
- Images dans des containers :
  - `bg-gray-50 rounded-3xl p-8`
- Typographie moderne :
  - Titre : `text-4xl font-bold`
  - Sous-titre : `text-lg text-gray-600`
- Buttons :
  - CTA principal : `bg-blue-600 hover:bg-blue-700 text-white px-6 py-3 rounded-xl font-semibold`
  - CTA secondaire : `border border-gray-300 bg-white hover:bg-gray-50 text-gray-900 rounded-xl px-6 py-3`

C’est le style visuel inspiré de **SwagUp**.

## 4. Principes de copywriting
Quand Copilot génère du texte :

- Toujours écrire **pour un RH / People Manager / Office Manager**, pas pour un marketeux.
- Mettre en avant :
  - “Plus de logistique interne”
  - “Plus de cartons dans les bureaux”
  - “Process simple”
  - “Vos collaborateurs remplissent eux-mêmes leurs infos”
  - “Livraison rapide”
  - “Qualité premium”
  - “Portail gratuit”
- Éviter :
  - Les phrases longues
  - Le ton trop américain / trop commercial
  - Le vocabulary trop technique
- Utiliser des verbes d’action :
  - Créer
  - Envoyer
  - Automatiser
  - Gérer
  - Expédier
  - Libérer du temps

## 5. CTAs recommandés
CTA principaux :
- “Créer mon portail”
- “Démarrer gratuitement”
- “Voir un exemple”

CTA secondaires :
- “Voir la démo”
- “Découvrir le catalogue”
- “Parler avec un expert”

## 6. Positionnement à garder en tête
Copilot doit toujours comprendre :

- OneStopMerch = votre **partenaire logistique invisible**.
- Vous gérez l’humain → OneStopMerch gère le matériel.
- Nous prenons en charge : production + stockage + emballage + expédition.
- Finis les Excel, les cartons au bureau, les démarches logistiques.
- Votre portail interne devient un **outil d’équipe**.
- L’expérience collaborateur est centrale (welcome packs premium).

## 7. À ne jamais faire
- Ne jamais inventer de fonctionnalités techniques non confirmées.
- Ne jamais promettre une automatisation complète (MVP = illusion + manuel).
- Ne pas écrire de paragraphes trop longs.
- Ne pas changer le positionnement RH/People.
